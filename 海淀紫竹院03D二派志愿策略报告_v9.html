📖 模块2：择校必知 — 5个核心概念通俗解读 概念版新增

以下内容专为"完全不懂小升初术语的家长"编写，用最通俗的语言解释报告中反复出现的关键概念。建议先读完这5个概念，再看后面的数据对比和策略建议。

🎓 概念1：校额到校 — "校内竞争，不跟全区学霸拼"

📌 本质：示范高中（人大附、北大附等名校）给每所初中预留的"保送名额"。孩子在初中校内排名靠前，就能通过这些名额进入好高中，不用跟全区几万人挤统招。
💡 举例：理工附中东校区有19个校额到校名额。如果你孩子在校内考前19名（且达到最低分数线），就能进人大附、北大附、清华附等名校高中。
⭐ 为什么重要：校额到校比例越高 = 校内竞争越小 = 进名校高中概率越大。这是选初中最核心的指标。
⚠️ 注意：校额到校是中考（初中升高中）的机制，不是小升初的。选初中时看这个指标，是为了3年后的升学优势。

🚀 概念2：1+3项目 — "初二直升高中，免中考"

📌 本质：初二下学期选拔优秀学生，直接进入4年制高中（1年衔接 + 3年高中），不参加中考。
💡 举例：首师大二附中有80个1+3名额（占20%），意味着初二结束时80个学生可以直接进入首师大二附高中部，完全跳过中考。流程：初二下报名 → 面试/考核 → 录取 → 初三开始在高中校就读。
⭐ 为什么重要：1+3名额越多 = 免中考的机会越大 = 初中阶段的"保险系数"越高。通常需要在校成绩排名靠前，通过学校自主考核。

⏰ 概念3：0.5+3项目 — "初三下提前进高中"

📌 本质：初三下学期（0.5年）提前进入高中校做初高衔接，然后直接读3年高中，也免中考。
💡 举例：十九中有27个0.5+3名额（占3.86%），初三下有27个学生可以提前进入十九中高中部。
⭐ 与1+3的区别：1+3是初二走（更早、名额更多），0.5+3是初三下走（更晚、名额更少）。0.5+3多在教育集团内部进行（如人大附集团、首师大附集团）。

📝 模块9：志愿排序建议

推荐排序（综合专业报告建议）

核心原则：03D学区7所学校整体均衡，没有绝对的天坑校。志愿排序的关键在于"一志愿冲还是稳"以及"二志愿拦截谁"。PDF5 03D报告

方案A：冲拦结合（适合风险偏好较高的家庭）

志愿序号推荐学校策略定位中签概率
一志愿首师大二附中 或 十九中学冲高≤40%
二志愿海淀实验中学 或 理工附中分校拦截视一志愿溢出情况
三志愿理工附中东校区 或 北外附中补位视前序溢出
四~七志愿剩余学校，末位填北京实验学校兜底高
方案A风险提示：一志愿冲高失败概率约60%，落签后大量家庭涌入二志愿，可能导致海实/理工附分二志愿也大塞车。做好三志愿以后才中签的心理准备。

方案B：稳拦结合（推荐多数家庭采用）

志愿序号推荐学校策略定位中签概率
一志愿理工东 / 海实 / 理工附分 3选1稳中求进≤80%
二志愿海实 / 理工附分 2选1拦截求捡漏视一志愿溢出
三志愿北外附中补位视前序溢出
四~七志愿剩余学校，末位填北京实验学校兜底高
方案B优势：一志愿竞争小（热度3的学校填报率~0%），中签概率显著提高。即便一志愿未中，二志愿仍有较好机会。

关键风险提示

1. 踩踏风险：首师大二附中（30%填报 / 32%中签）和十九中学（40%填报 / 41%中签）一志愿竞争最激烈。PDF5

2. 塞车风险：一派冲高落签的家庭大量进入二派，倾向于求稳妥，导致原本热度3的学校在二志愿阶段突然变热。

3. 北京实验兜底：虽然热度最低，但综合贯通培养比例全学区最高（58.25%），作为末志愿并不亏。PDF3/PDF5

4. 北外附中独特性：区独享校，只有紫竹院学区能进，小语种特色鲜明，适合有明确外语兴趣的家庭。PDF3/PDF4

⚙ 模块11：二派名额算法说明 v8新增

本模块解释二派名额的两层推算逻辑，帮助家长理解专业报告中35-60人的预测值是如何得出的。

11.1 第一层：全校二派总数推算

每所中学的全校二派招生总数，等于总招生容量减去其他通道已录取的人数：

// 第一层：全校二派总数 全校二派总数 = 总招生名额(班数 × 每班人数) - 一派(登记入学)名额 - 直升人数(对口小学直升比例 × 毕业生人数) - 公办寄宿(如有) - 黑盒子(政保 + 密电 + 点招，阳光招生后约5-12%)

11.2 第二层：片区分配

全校二派名额并非全部给一个学区，而是按比例分配给周边多个学区：

// 第二层：片区分配 03D片区二派名额 = 全校二派总数 × 03D片区分配比例
关键理解：专业报告预测的35-60人是03D片区的份额，不是全校二派总数。例如理工附中东校区全校二派可能有200+人，但分配给03D片区约60人（其余分配给羊坊店、八里庄等相邻学区）。
